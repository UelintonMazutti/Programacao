Bibliotecas Importantes em C - math.h e outras

Curso de Lógica de Programação com Linguagem C

1. Introdução às Bibliotecas em C

Bibliotecas em C são coleções de funções pré-definidas que expandem as capacidades da linguagem. Elas são incluídas usando a diretiva #include.

Principais bibliotecas padrão:

  • stdio.h - Entrada e saída padrão
  • math.h - Funções matemáticas
  • stdlib.h - Funções utilitárias gerais
  • string.h - Manipulação de strings
  • time.h - Manipulação de data e hora

2. A Biblioteca math.h

A biblioteca math.h contém funções matemáticas avançadas. Para usá-la, você precisa:

#include <math.h>
#include <stdio.h>

int main() {
    // Seu código aqui
    return 0;
}

Observação: Ao compilar, você pode precisar linkar a biblioteca matemática com a flag -lm:

gcc programa.c -o programa -lm

3. Principais Funções da math.h

Funções Trigonométricas:

Função Descrição Exemplo
sin(x) Seno de x (radianos) sin(3.14159/2) → ~1.0
cos(x) Cosseno de x (radianos) cos(0) → 1.0
tan(x) Tangente de x (radianos) tan(0.785398) → ~1.0

Funções Exponenciais e Logarítmicas:

Função Descrição Exemplo
exp(x) e elevado a x exp(1) → ~2.71828
log(x) Logaritmo natural (base e) log(2.71828) → ~1.0
log10(x) Logaritmo base 10 log10(100) → 2.0

Funções de Potência e Arredondamento:

Função Descrição Exemplo
pow(x, y) x elevado a y pow(2, 3) → 8.0
sqrt(x) Raiz quadrada de x sqrt(16) → 4.0
ceil(x) Arredonda para cima ceil(3.2) → 4.0
floor(x) Arredonda para baixo floor(3.9) → 3.0
fabs(x) Valor absoluto (float/double) fabs(-5.5) → 5.5

1. Faça um programa que calcule a hipotenusa de um triângulo retângulo, pedindo ao usuário os catetos

Fórmula: hipotenusa = √(cateto1² + cateto2²)

#include <stdio.h>
#include <math.h>

int main() {
    double cateto1, cateto2, hipotenusa;
    
    printf("Digite os dois catetos: ");
    scanf("%lf %lf", &cateto1, &cateto2);
    
    hipotenusa = sqrt(pow(cateto1, 2) + pow(cateto2, 2));
    
    printf("A hipotenusa é: %.2lf\n", hipotenusa);
    
    return 0;
}

2. Faça um programa que calcule as raízes de uma equação de segundo grau usando a fórmula de Bhaskara

Fórmula: x = [-b ± √(b² - 4ac)] / (2a)

#include <stdio.h>
#include <math.h>

int main() {
    float a, b, c, delta, x1, x2;
    
    printf("Digite os coeficientes a, b e c: ");
    scanf("%f %f %f", &a, &b, &c);
    
    delta = pow(b, 2) - 4 * a * c;
    
    if (delta < 0) {
        printf("Não existem raízes reais.\n");
    } else {
        x1 = (-b + sqrt(delta)) / (2 * a);
        x2 = (-b - sqrt(delta)) / (2 * a);
        printf("As raízes são: %.2f e %.2f\n", x1, x2);
    }
    
    return 0;
}

17. Faça um programa que calcule a área de um triângulo usando a fórmula de Heron

Fórmula: Área = √[s(s-a)(s-b)(s-c)], onde s = (a+b+c)/2

#include <stdio.h>
#include <math.h>

int main() {
    float a, b, c, s, area;
    
    printf("Digite os três lados do triângulo: ");
    scanf("%f %f %f", &a, &b, &c);
    
    if (a + b > c && a + c > b && b + c > a) {
        s = (a + b + c) / 2;
        area = sqrt(s * (s - a) * (s - b) * (s - c));
        printf("Área do triângulo: %.2f\n", area);
    } else {
        printf("Os lados não formam um triângulo válido!\n");
    }
    
    return 0;
}

#define G 9.8

20. Faça um programa que calcule o valor de e^x usando a série de Taylor (primeiros 10 termos)

Fórmula: e^x ≈ 1 + x + x²/2! + x³/3! + ... + x^9/9!

#include <stdio.h>
#include <math.h>

int main() {
    double x, ex = 1.0, termo = 1.0;
    int i;
    
    printf("Digite o valor de x: ");
    scanf("%lf", &x);
    
    for (i = 1; i <= 9; i++) {
        termo *= x / i;
        ex += termo;
    }
    
    printf("e^%.2lf ≈ %.6lf\n", x, ex);
    printf("Valor real: %.6lf\n", exp(x));
    
    return 0;
}

21. Faça um programa que gere 6 números aleatórios para uma loteria (entre 1 e 60 sem repetição)

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int numeros[6], i, j, repetido;
    
    srand(time(NULL));
    
    for (i = 0; i < 6; i++) {
        do {
            numeros[i] = rand() % 60 + 1;
            repetido = 0;
            for (j = 0; j < i; j++) {
                if (numeros[j] == numeros[i]) {
                    repetido = 1;
                    break;
                }
            }
        } while (repetido);
    }
    
    printf("Números da loteria: ");
    for (i = 0; i < 6; i++) {
        printf("%d ", numeros[i]);
    }
    printf("\n");
    
    return 0;
}

        printf("%d ", parte2[i]);
    }
    printf("\n");
    
    free(parte1);
    free(parte2);
    
    return 0;
}
