Módulo 1: Introdução à Programação com Python

Curso de Lógica de Programação com Python

1. Conceitos Fundamentais de Programação

Programar é escrever instruções que serão executadas por um computador.

Conceitos importantes:

  • Algoritmo: sequência de passos para resolver um problema.
  • Lógica de programação: raciocínio necessário para criar soluções computacionais.
  • Fluxograma: representação gráfica da lógica do algoritmo.
  • Pseudocódigo: forma textual simples de representar a lógica antes de programar.

2. Introdução à Linguagem Python

Python é uma linguagem de programação interpretada, de alto nível e com sintaxe simples e legível.

Características principais:

  • Fácil aprendizado e leitura
  • Tipagem dinâmica
  • Multiplataforma
  • Grande comunidade e ecossistema
  • Amplamente usada em ciência de dados, IA, web e automação

3. Estrutura de um Programa em Python

Um programa básico em Python possui esta estrutura:

# Comentários em Python começam com #

# Função principal (não é obrigatória como em C)
def main():
    # Instruções do programa
    print("Olá, mundo!")

# Verifica se este arquivo está sendo executado diretamente
if __name__ == "__main__":
    main()

Explicação:

  • def main():: define a função principal (opcional em Python)
  • print(): função para exibir mensagens na tela
  • if __name__ == "__main__":: padrão para executar o programa

4. Variáveis e Tipos de Dados

Em Python, as variáveis são criadas quando você atribui um valor a elas (tipagem dinâmica).

Principais tipos:

Tipo Descrição Exemplo
int Números inteiros idade = 30
float Números com ponto decimal altura = 1.75
str Texto (string) nome = "João"
bool Valores booleanos ativo = True

Exemplo em código:

def main():
    idade = 25
    salario = 2500.75
    nome = "Maria"
    ativo = True

    print(f"Idade: {idade}")
    print(f"Salário: {salario:.2f}")
    print(f"Nome: {nome}")
    print(f"Ativo: {ativo}")

if __name__ == "__main__":
    main()

5. Operadores Aritméticos e Lógicos

Aritméticos:

Operador Operação Exemplo
+ Adição a + b
- Subtração a - b
* Multiplicação a * b
/ Divisão a / b
// Divisão inteira a // b
% Módulo (resto) a % b
** Exponenciação a ** b

Relacionais:

Operador Descrição
== Igual a
!= Diferente de
> Maior que
< Menor que
>= Maior ou igual
<= Menor ou igual

Lógicos:

Operador Descrição Exemplo
and E (AND) a > 0 and b < 10
or Ou (OR) a == 0 or b == 1
not Negação (NOT) not (a == b)

Exemplo em código:

def main():
    a = 5
    b = 3

    print(f"Soma: {a + b}")
    print(f"Multiplicação: {a * b}")
    print(f"É igual? {a == b}")
    print(f"Maior que 0 e menor que 10? {a > 0 and a < 10}")

if __name__ == "__main__":
    main()

6. Entrada e Saída de Dados (input e print)

print() – Saída

def main():
    idade = 28
    print("Minha idade é:", idade)
    # Usando f-string (Python 3.6+)
    print(f"Minha idade é: {idade}")

if __name__ == "__main__":
    main()

input() – Entrada

def main():
    idade = input("Digite sua idade: ")
    print("Você digitou:", idade)
    
    # Convertendo para inteiro
    idade_int = int(input("Digite sua idade: "))
    print("No próximo ano você terá:", idade_int + 1)

if __name__ == "__main__":
    main()

Observações:

  • input() sempre retorna uma string
  • É necessário converter para outros tipos quando necessário (int(), float())
  • f-strings (Python 3.6+) permitem formatação fácil de strings
Exercícios

1. Faça um programa que escreva na tela "Olá, mundo"!

3. Faça um programa que leia a temperatura em Celsius e Converta para Fahrenheit

Fórmula: fahrenheit = (celsius * 9/5) + 32

celsius = float(input("Digite a temperatura em Celsius: "))
fahrenheit = (celsius * 9/5) + 32
print(f"Temperatura em Fahrenheit: {fahrenheit:.2f}")

4. Faça um programa que calcule a área do Círculo. Pedindo pro usuário informar o raio do circulo.

Para fazer o exercício você vai precisar declarar o valor de PI e fazer a seguinte Fórmula: PI * raio * raio

import math

raio = float(input("Digite o raio do círculo: "))
area = math.pi * raio ** 2
print(f"A área do círculo é: {area:.2f}")

11. Faça um programa que imprima o volume de uma esfera, para isso peça ao usuário o Raio da Esfera

15. Faça um program para calcular o delta de uma equação de segundo grau

b = float(input("Digite o coeficiente b: "))

print(f"O valor de delta é: {delta:.2f}")

20. Faça um programa que leia um ano e valide se esse ano é Bissexto ou não

Regra: Um ano é bissexto se for divisível por 4 e, se também for divisível por 100, precisa ser divisível por 400.

ano = int(input("Digite um ano: "))
if (ano % 4 == 0 and ano % 100 != 0) or (ano % 400 == 0):
    print("O ano é bissexto.")
else:
    print("O ano não é bissexto.")
