Módulo 1: Introdução à Programação e Lógica Básica

Curso de Lógica de Programação com Linguagem C

1. Conceitos Fundamentais de Programação

Programar é escrever instruções que serão executadas por um computador.

Conceitos importantes:

  • Algoritmo: sequência de passos para resolver um problema.
  • Lógica de programação: raciocínio necessário para criar soluções computacionais.
  • Fluxograma: representação gráfica da lógica do algoritmo.
  • Pseudocódigo: forma textual simples de representar a lógica antes de programar.

2. Introdução à Linguagem C

A linguagem C é estruturada, rápida e eficiente. É muito usada em sistemas embarcados, jogos e sistemas operacionais.

Características principais:

  • Portável e performática
  • Controle total de memória
  • Base para linguagens como C++, Java e Python

3. Estrutura de um Programa em C

Um programa básico em C possui esta estrutura:

#include <stdio.h>


int main() {
    // Instruções do programa
    return 0;
}

Explicação:

  • #include <stdio.h>: inclui a biblioteca de entrada e saída.
  • int main(): é a função principal.
  • { }: delimitam o bloco de código.
  • return 0;: indica finalização bem-sucedida.

4. Variáveis e Tipos de Dados

Variáveis armazenam valores que podem mudar durante a execução.

Principais tipos:

Tipo Descrição Exemplo
int Números inteiros int idade = 30;
float Números com ponto decimal float altura = 1.75;
char Caractere char letra = 'A';
double Decimal com maior precisão double pi = 3.14159;

Exemplo em código:

#include <stdio.h>


int main() {
    int idade = 25;
    float salario = 2500.75;
    char sexo = 'M';
    double pi = 3.141592;

    printf("Idade: %d\n", idade);
    printf("Salário: %.2f\n", salario);
    printf("Sexo: %c\n", sexo);
    printf("PI: %.5lf\n", pi);

    return 0;
}

5. Operadores Aritméticos e Lógicos

Aritméticos:

Operador Operação Exemplo
+ Adição a + b
- Subtração a - b
* Multiplicação a * b
/ Divisão a / b
% Módulo a % b

Relacionais:

Operador Descrição
== Igual a
!= Diferente de
> Maior que
< Menor que
>= Maior ou igual
<= Menor ou igual

Lógicos:

Operador Descrição Exemplo
&& E (AND) a > 0 && b < 10
|| Ou (OR) a == 0 || b == 1
! Negação (NOT) !(a == b)

Exemplo em código:

#include <stdio.h>


int main() {
    int a = 5, b = 3;

    printf("Soma: %d\n", a + b);
    printf("Multiplicação: %d\n", a * b);
    printf("É igual? %d\n", a == b);
    printf("Maior que 0 e menor que 10? %d\n", a > 0 && a < 10);

    return 0;
}

6. Entrada e Saída de Dados (printf e scanf)

printf() – Saída

#include <stdio.h>


int main() {
    int idade = 28;
    printf("Minha idade é: %d\n", idade);
    return 0;
}

scanf() – Entrada

#include <stdio.h>


int main() {
    int idade;
    printf("Digite sua idade: ");
    scanf("%d", &idade);
    printf("Você digitou: %d\n", idade);
    return 0;
}

Tabela de Formatos

Formato Tipo
%d Inteiro
%f Float
%lf Double
%c Caractere
%s String (texto)

⚠️ Use o & antes da variável no scanf() para indicar o endereço de memória.

15. Faça um program para calcular o delta de uma equação de segundo grau

Peça ao usuário 3 números (a, b, c) e use a fórmula: delta = b * b - 4 * a * c

#include <stdio.h>

int main() {
    float a, b, c, delta;

    printf("Digite os coeficientes a, b e c da equação (ax² + bx + c): ");
    scanf("%f %f %f", &a, &b, &c);

    delta = b * b - 4 * a * c;

    printf("O valor de delta é: %.2f\n", delta);

    return 0;
}

20. Faça um programa que leia um ano e validde se esse ano é Bissexto ou não

Regra: Um ano é bissexto se for divisível por 4 e, se também for divisível por 100, precisa ser divisível por 400.

#include <stdio.h>

int main() {
    int ano;
    printf("Digite um ano: ");
    scanf("%d", &ano);
    if ((ano % 4 == 0 && ano % 100 != 0) || (ano % 400 == 0)) {
        printf("O ano é bissexto.\n");
    } else {
        printf("O ano não é bissexto.\n");
    }
    return 0;
}
